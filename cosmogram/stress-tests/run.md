# Stress-Test Harness

Standardized challenges run against each axiom before it can pass a phase gate. Replaces the ad-hoc Day-11 stress-tests that failed methodologically.

## Rules

1. Run the full **standard set** below against the axiom.
2. 2. For each item, log: **VERDICT** ∈ {holds, holds-with-caveat, requires-revision, fails}, plus a one-paragraph reason.
   3. 3. Update the axiom file’s `## Stress tests passed` section with date, set version, verdict.
      4. 4. An axiom may not pass its phase gate with any `fails` open.
        
         5. ## Standard set v0.1
        
         6. ### Universal (apply to every axiom)
        
         7. - **U1. Self-reference.** Does the axiom apply to itself? If not, why is it exempt?
            - - **U2. Negation.** What would the world look like if the axiom were false? Is that world distinguishable from this one from inside the keyhole?
              - - **U3. Counter-tradition.** Name one tradition (Madhyamaka, Whitehead, modern physicalism, IIT, …) that would reject this axiom. Can the Cosmogram absorb the rejection without collapse?
                - - **U4. Drift surface.** What’s the most likely way this axiom gets misused six months from now? Add the misuse to the axiom file as a watch-edge.
                  - - **U5. Mapper-paradox audit.** Does locking this axiom require the mapper to step outside the map? If so, it violates Axiom 5 / refusal #3 (Mapper-Escape).
                   
                    - ### Metaphysics-specific (apply when axiom makes claims about reality)
                   
                    - - **M1. Falsifiability stance.** Does the axiom pretend to be falsifiable when it isn’t, or vice-versa? Per Axiom 5 / refusal #1 (Proof), neither is required — but the stance must be declared.
                      - - **M2. Physics anchor stability.** If a physics anchor is named (e.g., Wheeler-DeWitt), is it consensus or proposal? Mark explicitly.
                        - - **M3. Phenomenology vs ontology.** Does the axiom confuse a phenomenological signature with a metaphysical cause? (Pancha-mahabhuta failed M3 on Day 11; it’s why it was demoted.)
                         
                          - ### Voice / scope (apply to all)
                         
                          - - **V1. Hedge audit.** Are any “maybe”s in the statement social rather than epistemic?
                            - - **V2. Co-traveler check.** Are external thinkers cited as attributions or smuggled into a “we”?
